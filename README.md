# Dostaffkin — Доставка посылок

Веб-приложение для оформления и отслеживания доставки посылок с построением маршрута на карте.

## Основные возможности

- **Оформление доставки**  
  Выбор пунктов отправления и назначения (с подсказками адресов), размера посылки, скорости доставки. Автоматический расчёт стоимости и срока доставки на основе расстояния.

- **Отслеживание отправления**  
  Поиск по номеру заказа, просмотр маршрута и статусов доставки.

- **Главная страница**  
  Информационный блок с основными преимуществами и кнопками перехода к ключевым разделам.

## Технологический стек

- **Frontend**: Angular 17 (сигналы, реактивные формы, роутинг, HttpClient)
- **Карты и подсказки**: Yandex Maps API 2.1
- **Стилизация**: CSS, адаптивная вёрстка (мобильные устройства)
- **HTTP-клиент**: взаимодействие с внешним API
- **Язык**: TypeScript

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите локальный сервер разработки:
   ```bash
   npm start
   ```
   или
   ```bash
   ng serve
   ```

3. Откройте в браузере `http://localhost:4200`

## API

Приложение использует тестовое API:
- `POST https://testologia.ru/delivery/create` – создание заявки
- `GET https://testologia.ru/delivery/info?id={id}` – получение информации о доставке

Ключ для Yandex Maps уже встроен в `index.html`.
